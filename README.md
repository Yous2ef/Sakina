## 🛠️ التقنيات المستخدمة

| التقنية             | الوصف                            |
| ------------------- | -------------------------------- |
| **React 19**        | واجهة المستخدم التفاعلية         |
| **Vite 7**          | أداة بناء فائقة السرعة           |
| **TailwindCSS 4**   | تنسيق حديث ومرن                  |
| **Framer Motion**   | رسوم متحركة سلسة                 |
| **PWA**             | تطبيق ويب تقدمي يعمل بدون إنترنت |
| **Umami Analytics** | إحصائيات تحترم الخصوصية          |
| **GitHub Pages**    | استضافة مجانية وسريعة            |

## 🚀 التشغيل

```bash
# تثبيت المكتبات
npm install

# تشغيل بيئة التطوير
npm run dev

# بناء للإنتاج
npm run build

# نشر على GitHub Pages
npm run deploy
```

## 📁 هيكل المشروع

```
src/
├── components/     # المكونات (DhikrCard, Header, BottomNav...)
├── pages/          # الصفحات (Home, MorningDhkar, EveningDhkar...)
├── data/           # البيانات (أذكار الصباح، المساء، مفاتيح الفرج)
├── hooks/          # Custom Hooks (useProgress, useTheme...)
├── context/        # Context API (Theme, Settings, Progress)
├── config/         # ملفات التكوين
├── services/       # الخدمات (الإشعارات، المشاركة، التخزين، الإحصائيات)
└── utils/          # الأدوات المساعدة
```

## 📈 الإحصائيات والخصوصية

نستخدم **[Umami Analytics](https://umami.is)** - بديل مفتوح المصدر يحترم الخصوصية:

- ✅ لا يتتبع المستخدمين شخصياً
- ✅ لا يستخدم Cookies
- ✅ لا يجمع بيانات شخصية
- ✅ متوافق مع GDPR
- ✅ البيانات مجهولة تماماً

الإحصائيات تشمل: عدد الزوار، المشاهدات، التثبيتات - بدون أي معلومات شخصية.

## 📖 مصادر الأذكار

- صحيح البخاري
- صحيح مسلم
- سنن الترمذي
- سنن أبي داود
- مسند أحمد
- كتاب "مفاتيح الفرج" للشيخ سالم أبو الفتوح

## 🎯 الاستخدام

1. افتح التطبيق
2. اختر القسم المناسب (صباح/مساء/تسبيح/مفاتيح)
3. اضغط على البطاقة لزيادة العداد
4. تابع تقدمك من خلال الدائرة التقدمية

## 📲 التثبيت كتطبيق

التطبيق يدعم PWA - يمكنك تثبيته على هاتفك:

1. افتح التطبيق في المتصفح
2. اضغط على "إضافة إلى الشاشة الرئيسية"
3. استمتع بالتطبيق بدون إنترنت!
